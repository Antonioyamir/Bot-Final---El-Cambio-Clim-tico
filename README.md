# Bot Final El Cambio Climatico
Este es un bot de discord en donde aprenderas nuevas cosas sobre el cambio climático! conoceras, muchas catastofres, y  las causas y consecuencias, y por ultimo, Como solucionarlo 😀

Comandos: 🌲
/hola
/clima
/ayuda
/masinfo
/inundaciones
/tornados
/sequias
/catastrofes
/cambioclimatico
/causas
/efectos
/soluciones
/calentamiento
/adios

Como instalar?: 🍃
1. Crea tu token o bot
2. Instalar discord (libreria): pip install discord
3. Instalar requests: pip install requests
4. Instalar pyttsx3: pip install pyttsx3
5. Ejecutar Archivo.
Eso es todo! y pones los comando.
